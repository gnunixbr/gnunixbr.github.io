Ativando as opções de desenvolvedor

O Android possui as “Opções de desenvolvedor”, um menu especial com configurações relevantes para usuários avançados e desenvolvedores. Muitas de tais opções vieram do Android e o CyanogenMod trouxe algumas modificações. Por padrão, este menu é oculto, mas pode ser ativado com s seguintes passos:

  1. Em “Configurações”, clique em “Sobre o telefone” (ou “Sobre o tablet”).
  2. Clique sete vezes em “Número da versão”. Na sétima vez, você será notificado de que “Você ativou as configurações de desenvolvedor!”.
  3. Retorne para o menu “Configurar”. As Opções de desenvovedors devem estar visíveis agora.
 

Nota:

A necessidade de se ativar manualmente as Opções de desenvolvedor foi introduzida pelo Android 4.1 (CyanogenMod 10.1).

Ativando a depuração USB

As técnicas de depuração mais usadas para dignosticar problemas no Android são Logcat e os relatórios de kernel (dmesg). O primeiro passo para usar tais ferramentas é ativar a depuração USB, daí você poderá gerar e transferir relatóorios dos seu dispositivo para o seu computador. A depuração USB possui inúmeros usos, para tanto, visite a página sobre adb.

Para ativar a depuração USB:

  1. Ative as Opções de desenvolvedor seguindo as instruções constantes na seção anterior
  2. “Configurar” > “Opções de desenvolvedor” > “Depuração android”

Gerneciando o superusuário (root)

Desde o CyanogenMod 12, opções de root são ativadas nas Opções de desenvolvedor. Procure a opção “Acesso superusuário". Por questões de segurança, o Acesso superusuário vem desativado por padrão. O acesso superusuário para aplicativos são controlados através do menu “Configurar” > “Privacidade” > “Proteção à privacidade”. Clique num aplicativo listado para permitir/negar o acesso superusuário.
