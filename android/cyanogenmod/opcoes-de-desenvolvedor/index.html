Ativando as Opções de desenvolvedor

O Android possui asOpções de desenvolvedor, um menu especial com configurações relevantes para usuários avançados e desenvolvedores. Muitas das opões vieram do Android e o CyanogenMod acrescentou algumas. Por padrão, este menu é oculto, mas pode ser ativado como se segue:

  1. No menu "Configurar", selecione "Sobre o telefone" (or "Sobre o tablet").
  2. Clique sete vezes em "Número da versão". Na sétima vez, aparecerá uma notificação dizendo que “Você ativou as opções de desenvolvedor!”
  3. Retorne ao menu "Configurar". As Opções de desenvolvedor agora devem estar visíveis.

Observação:

A necessidade de ativar manualmente as Opções de desenvolvedor foi introduzida pelo Andoid 4.1 (CyanogenMod 10.1).

Ativando a Depuração USB

As técnicas mais comuns de depuração usadas para disgnosticar problemas no Android são o Logcat e os relatórios de kernel(dmesg). O primeiro passo para usar tais ferramentas é ativar a depuração USB e daí você poderá gerar e trazer relatórios de seu dispositivo para seu computador. A depuração USB possui diversas utilidades. Para tanto, visite a página sobre adb.

Ativando a depuração USB:

  1. Ative as Opções de desenvolvedor seguindo as instruções constantes na seção anterior
  2. Configurar > Opções de desenvolvedor > Depuração Android

Gerenciando o super-usuário(root)

Desde o CyanogenMod 12, opções de root são ativadas através das Opções de desenvolvedor. Econtre a opção “Acesso superusuário”. Por questões de segurança, o Acesso superusuário vem desabilitado por padrão. Permissões de super-usuário para aplicativos são controladas em “Proteção à privacidade”, nas opções de Privacidade do menu “Configurar”. Clique num aplicativo para ativar/desativar o acesso de super-usuário.
