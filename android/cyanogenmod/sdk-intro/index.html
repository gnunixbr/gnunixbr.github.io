O que é Android SDK?

O Android SDK é um kit de desenvolvimento de software fornecido pelo Google para Windows, Mac OS X, e Linux. O Android SDK foi concebido para ajudar desenvolvedores a criar, testar, e depurar seus aplicativos Android. Para tanto, o SDK fornece documentação, exemplos de códigos, máquinas virtuais Android para a execução de aplicativos, códigos estruturais (framework), suporte para ambientes de desenvolvimento, ferramentas de depuração, e muito mais.

De especial proveito para os usuários finais que estão interessados em instalar e executar sistemas operacioanis alternativos (como o CyanogenMod), existem das ferramentas em particular:

  • adb -- o Android Debug Bridge, que fornece uma ponte entre computador e dispositivo, permitindo "empurrar" arquivos do computador para o dispositivo, "puxar" arquivos do dispositivo para o computador, e até mesmo interagir com o dispositivo através de uma interface por linha de comando. adb é uma ferramenta bastante útil que você deverá usar frequentemente se estiver interessado em explorar a fundo o seu dispositivo.
  • fastboot -- fastboot é uma outra ferramenta usada para gravar novos sistemas operacionais, limpar partições, desbloquear alguns dispositivos, e muito mais.

Como eu obtenho/instalo o Android SDK?

You Você pode obter as intruções de instalação para a sua plataforma pelo site oficial do Android SDK . Se você não está interessado no desenvolvimento de aplicativos Android, e apenas pretende usar o adb e o fastboot, leia a seção entitulada OBTER O SDK PARA UM IDE EXISTENTE e baixe o pacote de ferramentas do SDK.

 

Dica importante

If Se você planeja escrever aplicativos Android, talve você queira dar uma olhada no Android Studio, um IDE (Integrated Development Environment) livre, que pode te orientar no desenvolvimento dos seus programas. Você pode pensar num IDE, em parte, como um editor de texto ultra sofisticado, que organiza odos os arquivos que compõem seu alicativo. Ele também pode também fazer sugestões de código, fornecer ferramentas para edição de interfaces de suário, conferir a sintaxe, bem como ajudar a executar e depuar aplicativos em diferentes versões do Android.
